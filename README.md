# PySSL

## Flatform:
PySSL provided tool and dynamic library for windows 10 and above.

## Python version:
Supported for python version 3.7 and above.

## Install in Window
~~~terminal
>> pip install pyssl-package
~~~

## How to use it?
~~~terminal
>> python -m pySSL
~~~

## Security algorithms for generate key:
1. RSA Algorithms.
2. hash AES256/512
